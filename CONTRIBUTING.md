# Contributing

This repository is a time-bounded capstone project.

1. Create a focused branch.
2. Add or update tests for behavioural changes.
3. Do not add real personal data, client policies, credentials, or proprietary AUREX SICOS material.
4. Keep MCP tools read-only unless the architecture and security documents are formally revised.
5. Run `pytest` and `ruff check .` before opening a pull request.
