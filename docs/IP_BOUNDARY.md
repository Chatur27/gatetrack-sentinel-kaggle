# Public/private intellectual-property boundary

## Included in this public capstone

- Synthetic visitor records and fictional policies.
- Generic visitor-routing rules created for the competition.
- Read-only MCP policy tools.
- ADK demonstration agent.
- Generic human-review and audit patterns.

## Explicitly excluded

- AUREX SICOS scoring, readiness, risk-index, or KRI methodology.
- Client policies, procedures, audit findings, or personal data.
- Commercial GateTrack source code and product roadmap.
- Live identity, sanctions, biometric, or regulatory integrations.
- Private pricing, business strategy, credentials, or deployment keys.

The public repository should remain independently understandable and reproducible without exposing proprietary commercial assets.
