\# GateTrack Sentinel — Demo Video Script



\## 0:00 — Opening

This is GateTrack Sentinel, a human-governed AI control prototype by AUREX Sentinel Labs.



\## 0:20 — Problem

Visitor-risk decisions can become opaque when AI is used without controls. GateTrack Sentinel keeps the AI bounded, auditable and subordinate to human authority.



\## 0:45 — Visitor request

Submit a synthetic visitor request and show deterministic routing.



\## 1:20 — Security test

Submit a prompt-injection style request. Show that the system blocks it before model review.



\## 1:50 — Human review

Submit a sensitive case and show that the system requires human authority before finalisation.



\## 2:30 — Audit evidence

Open the audit timeline and show that the decision path is recorded.



\## 3:00 — Proof \& replay

Export or verify the proof packet, then run deterministic replay.



\## 3:40 — Loop control

Show bounded loop contracts and explain that every loop has limits.



\## 4:10 — Evaluation

Show deterministic evaluation and reliability checks.



\## 4:40 — Closing

GateTrack Sentinel demonstrates agentic AI with bounded execution, tool grounding, human oversight, and proof-carrying decisions.
